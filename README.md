# Typesafe's Play Framework

### Requisites

- [Java 8]

### Installation

```sh
$ brew install typesafe-activator
$ git clone [this repo] PlayFramework24
$ cd PlayFramework24
$ activator run 
```
The application is now available at localhost:9000 in your browser
